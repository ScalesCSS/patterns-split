Split Pattern for Scales

This page includes the default split pattern styles

Split usage

The Split pattern can be used on any elements but works best in a <dl> or <ul/ol>. For Example:

Default styles on a <dl>

Google
google.com
Wikipedia
wikipedia.org
CNN
cnn.com

Default styles on a <ul>

  • Google google.com
  • Wikipedia wikipedia.org
  • CNN cnn.com
